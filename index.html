Welcome to QwebView.

An easy interface to interact between javascript, python D3.js and qWebView Run the file by using python qwebiewtest.py Make the changes in the htmlFile.py If you want to test the html file simply copy paste the html code onto the testHtml and double click it.

Features

1) Very useful to add d3.js ibraries onto the Python widget. \n 2) A Game Changer for interacting python with web based objects, primarily javascript \n 3) Great addition to the libraries\n
